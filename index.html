1.해양 탐사를 위한 수중 영상 객체 분할 모델 -학사졸업논문

GitHub 링크 보기

목표

수중 영상 데이터에서도 높은 분할 성능을 내는 객체 분할 모델을 개발한다.
색상 왜곡과 조명 불균일성에도 강건한 성능을 목표로 한다.

문제점

공개된 수중 데이터셋이 적어 일반화가 어렵다.
적은 데이터셋으로 인해 Fine-tuning 과정에서 기존 사전학습 가중치가 왜곡될 수 있다.

해결법

기존 가중치를 그대로 유지한 상태에서 ControlNet의 제로-컨볼루션 기반 미세조정 기법을 적용한다.
이를 통해 원래 성능은 보존하면서 수중 도메인 특성을 반영한다.

결과

기존 모델보다 높은 성능의 모델 생성 자세한 내용은 github링크 참조
